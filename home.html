Plantlister

What is it?

Plantlister is a full-stack website dedicated to plant enthusiasts, offering a platform to buy and sell a wide variety of seeds, seedlings, plants, trees, and more.

What can a user do?

-User Registration and Authentication: Users can create accounts, log in, and manage their profiles.

-Create, Edit, Update and Delete Posts: Users can create new posts to sell plants, edit, update, and delete them when necessary.

-Responsive Design: The website is designed to be fully responsive, providing a seamless user experience across various devices, including tablets and mobile phones.

FuelSwitcheroo

What does it do?

This project utilizes Mapbox API and The National Renewable Energy Laboratory's developer network API to search for alternative fuel gas stations within the users marker area or whatever they search. The project was built using HTML, CSS, and Javascript and involved retrieving and displaying information about the nearest stations using the Alternative Fuels API, which returns JSON data.

Fun Fact:

This project received two awards from RowdyHacks, a hackathoon hosted by the Univerisy of Texas at San Antonio.
