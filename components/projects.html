Movies List Project

What does it do?

The Movies Project is a website that allows a user to add, edit, and delete movie list items. The list of movies is shown to the users as cards. The user is allowed to add a movie by typing in a title, genre, director, and a rating. The user can also edit an existing movie. If the user wants, they can also delete movies from the list.

Technologies used

  • Javascript
  • HTML
  • CSS
  • Bootstrap
  • JQuery
